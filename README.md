## Description

This is the backend for the FindNext project. FindNext is a simple social network built to people get to know new places to visit and explore.

## :fire: Technologies

This project was built using the following techs:

- [Nest](https://github.com/nestjs/nest)
- TypeScript.
- GraphQL (Apollo Server)
- PostgreSQL RDS (AWS Service)
- ElasticBeanStalk (AWS Service)
- S3 (AWS Service)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
